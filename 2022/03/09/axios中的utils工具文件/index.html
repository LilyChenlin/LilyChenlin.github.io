什么是Buffer

参考

Buffer 缓冲区

存在原因：

  1. JavaScript 语言自身只有字符串数据类型，没有二进制数据类型。
  2. 但在处理像TCP流或文件流时，必须使用到二进制数据。因此在 Node.js中，定义了一个 Buffer 类，该类用来创建一个专门存放二进制数据的缓存区。
